Anime Koyomi

Anime Koyomi is a web app that lets you easily add anime simulcast times to your Google Calendar.

Simply select the shows you want to follow (you can mix multiple sources), select the calendar you want the events to be created in, and go.

Timezone detection and conversion is handled automatically, but you can choose a custom timezone if you think something is off.
